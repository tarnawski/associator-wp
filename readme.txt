== Installation ==

= Install Via Plugin Repository =
1. Log into WordPress admin for your site
2. Go to 'Plugins -> Add New' and choose 'Search'
3. Search for Associator and click 'Install Now'
4. Activate the plugin through the 'Plugins' menu

= Install Manually using FTP =
1. Upload the complete `associator` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
